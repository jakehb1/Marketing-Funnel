## 💡 Usage Tips

### Quick Copy
Most content can be used directly. Just:
- Replace `[Name]` with recipient name
- Replace `[LINK]` with tracking URL
- Replace `[CTA]` with appropriate button text

### A/B Testing
Each type has 10+ variations. Use them to:
- Test different headlines
- Compare emotional angles
- Measure platform performance
- Optimize for your audience

### Customization
Templates use placeholder format:
- `{stage_name}` → Insert funnel stage
- `{emotional_intent}` → Reference the emotional hook
- `[Name]` → Recipient name
- `[LINK]` → URL/tracking code

### Quote Usage
Quotes are theme-organized. Use them to:
- Add credibility to copy
- Support claims about results
- Build authority in social posts
- Fill objection handlers with proof
