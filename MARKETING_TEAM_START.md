## Main Functions

### Upload a Video
1. Click "Video Manager" in sidebar
2. Click "Upload & Manage Videos" card
3. Drag your video into the upload zone (or click to select)
4. Wait for transcription to complete (1-3 minutes)
5. Video goes to Charlie for approval
6. Once approved, agents can use the transcript for ad copy

### View Content Library
1. Click "Content" → "Content Library"
2. You'll see 2,407 production-ready marketing assets
3. Everything organized by funnel and stage
4. Includes: emails, ad copy, landing pages, SMS, scripts, etc.

### View Funnels
1. Click "Dashboards" → "Funnels"
2. Click "Visual Strategy" to see all 5 funnels
3. Click any stage to see workflow details

### Monitor Approvals
1. Share approval link with Charlie: https://...railway.../approval
2. Charlie uses PIN: 12345
3. Charlie approves/rejects videos
4. You see approval status in real-time

## Common Tasks

### "I want to create an ad"
1. Go to Content Library
2. Find stage + copy variant for your funnel
3. Pick your favorite variation
4. Copy it to your ad platform (Meta, Google, TikTok, etc)
5. Agents will handle optimization

### "I need a new email template"
1. Check Content Library first
2. If not there, ask agents to generate one
3. They'll use existing videos + approved copy
4. Submit to Charlie for approval
5. Add to library once approved

### "Video transcription failed"
1. Check the video file (must be MP4, MOV, or WebM)
2. File size should be under 500MB
3. Try uploading again
4. Contact Jake if it keeps failing

### "I forgot Charlie's approval PIN"
1. Contact Jake
2. PIN is set in Railway environment variables
3. Can be changed anytime
